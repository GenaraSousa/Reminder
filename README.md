# Reminder
Aplicativo mobile para auxilio de aprendizado de conteúdos através de repetição e memorização.

## Tecnologias utilizadas no projeto

- React native
- Expo gerenciado
- React native async storage
- Notifications
- Datetimepicker
- React navigation
- Expo notification
- Expo fonts
- Expo google fonts
